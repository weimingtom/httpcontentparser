Main Features

Block webiste

我们提供了两个列表供您使用，一个是黑名单，所有在其中的网站都不能被访问。 第二个是白名单，这个名单内的网站是您认为比较安全的，可以供孩子访问。 而且，如果您希望加强对访问网站的管理，你可以使孩子只能访问某些网站。这会让您更加放心。

Block search word

控制孩子对搜索引擎的使用。现在搜索引擎已经十分强大，几乎能够或许任何想要的信息。但是有些信息如色情暴力等等，是您不希望孩子获得的。 这时候你就可以通过使用“Block Search word”功能，组织一些关键词的搜索。

Website Log

记录使用互联网的历史，包括访问过的网站、浏览过的图片、搜索过的词汇。

Screen Snapshot

定时自动记录屏幕。 而且提供了自动清理功能，您可以制定一个时间间隔我们会每个一段时间清理一次，为您节省宝贵的存储空间。

Eyesight Protection

游戏时候您希望您的孩子学习，不希望他们上网，此时“Online time contrl”就可以起作用了。 您可以通过它指定孩子在什么时候不能上网。在此期间，任何应用程序都无法访问网络。

Online time control

游戏时候您希望您的孩子学习，不希望他们上网，此时“Online time contrl”就可以起作用了。 您可以通过它指定孩子在什么时候不能上网。在此期间，任何应用程序都无法访问网络。

Application Control

有些应用程序您不希望孩子单独使用，如一些聊天工具、游戏等等。这时候您可以使用此功能，禁止这些应用程序的启动。 如果此功能被启用，所有保存在列表中的应用程序都无法被启动。
